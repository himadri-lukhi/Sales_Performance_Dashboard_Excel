# 📊 Sales Performance Dashboard | Microsoft Excel

## 📌 Project Overview

This project presents an interactive Sales Performance Dashboard built using Microsoft Excel. The objective was to analyze sales data, identify key business trends, and create a dashboard that enables quick decision-making through KPIs and interactive visualizations.


## 📂 Dataset

**Source:** Kaggle

**Dataset Link:**
https://www.kaggle.com/datasets/kyanyoga/sample-sales-data


## 🎯 Objectives

- Clean and prepare raw sales data.
- Analyze sales performance across products and countries.
- Track monthly sales trends.
- Build an interactive dashboard using Excel.
- Present key business insights through KPIs and visualizations.


## 🛠 Tools & Features Used

- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Custom Formatting
- Dashboard Design


## 📈 Key Performance Indicators (KPIs)

- 💰 Total Revenue
- 📦 Total Orders
- 👥 Total Customers
- 🚚 Total Quantity Sold
- 💳 Average Order Value


## 📊 Dashboard Visualizations

- Revenue by Product Line
- Monthly Sales Trend
- Revenue by Country
- Revenue by Deal Size
- Order Status Distribution
- Interactive Country Slicer


## 🔍 Key Insights

- Classic Cars generated the highest revenue among all product lines.
- Trains recorded the lowest revenue contribution.
- Medium-sized deals accounted for the largest share of total revenue.
- Sales show noticeable fluctuations across different months.
- The interactive slicer enables quick country-wise sales analysis.
